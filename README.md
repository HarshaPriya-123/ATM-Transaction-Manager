# 💳 ATM Web Service Application

A simple and user-friendly **ATM Web Application** that allows users to perform basic banking operations such as balance inquiry, deposits, withdrawals, and transaction history through an interactive web interface.


## 🌐 Live Demo

🔗 https://atmwebservice.netlify.app/


## ✨ Features

- 🔐 Secure user authentication (PIN/Login simulation)  
- 💰 Check account balance  
- ➕ Deposit money  
- ➖ Withdraw cash  
- 📜 View transaction history  
- ⚡ Fast and responsive interface  
- 🌐 Fully browser-based (no installation required)  


## 🛠️ Technologies Used

- **Frontend:** HTML5, CSS3, JavaScript  
- **Platform:** Netlify (for deployment)  


## 🚀 How to Run the Project

1. Clone or download the repository  
2. Open the project folder  
3. Double-click `index.html`  
4. The application will run in your browser  


## 📱 Use Cases

- Learning basic banking system workflow  
- Understanding transaction handling logic  
- Practicing frontend development skills  
- Demonstrating projects for internships  


## 🎯 Purpose of the Project

This project was developed to simulate real-world ATM banking operations and to strengthen frontend development skills, including user interaction, logic building, and UI design.
